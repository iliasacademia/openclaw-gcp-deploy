Connect Telegram

OpenClaw uses a Telegram bot to receive your messages and reply with AI-powered responses. You'll create a private bot in under a minute.

How to get your bot token

  1. Open Telegram and search for @BotFather (blue tick ✓)
  2. Tap Start, then send the command /newbot
  3. Choose a display name — e.g. My Claw Agent
  4. Choose a username ending in bot — e.g. myclawagent_bot
  5. BotFather replies with your token — copy it (looks like 123456789:ABCdef…)

✅

You're all set!
